**Одностраничный чат.**

Реализация заданий в рамках обучения в Яндекс.Практикум.

Как построить проект:

`npm install`

`npm run construct`

Как запустить проект?

`npm run start`

Как запустить тесты:

`npm run test`

Как проверить стили:

`npm run check-style`

Как проверить стили кода:

`npm run check-code`
